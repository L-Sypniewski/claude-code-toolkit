# AGENTS.md Documentation

Machine-readable guide for AI coding tools. Provides explicit setup, conventions, and workflows.

## Structure

**Required**: Title/metadata (Stack, Principles), Project Overview, Repository Structure, Key Commands, Quality Gates (build/test/lint standards).
**Optional**: Coding Conventions, Testing Guidelines.

## Best Practices

- Be explicit: Use exact commands (e.g., "npm install && cp .env.example .env")
- Keep updated when structure/conventions change
- Start with essentials: Stack, Overview, Structure, Commands
